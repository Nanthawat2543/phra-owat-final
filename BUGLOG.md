## รายการบัค/คำขอ

### 1. แก้ลำดับข้อมูลป็อปอัป "พระโอวาทแนะนำวันนี้" (ความสำคัญ 3)
เรียงใหม่: ① ประโยคพระโอวาท ② พระนามสิ่งศักดิ์สิทธิ์ ③ ณ สถานที่ ④ วันที่ ⑤ ปุ่ม "อ่านพระโอวาท"
เอาออก: ปุ่ม "ปิด" (มีกากบาทแล้ว) / ลูกศรที่ปุ่มต้นฉบับ / หัวเรื่องแบดจ์ "พระโอวาทแนะนำวันนี้"
→ ไฟล์: `src/pages/Home.tsx` (modal)

### 2. [เสนอ] เปลี่ยนคำ "พระโอวาทแนะนำวันนี้" → "พระโอวาทวันนี้"
→ `src/pages/Home.tsx`

### 3. [เสนอ] ประโยคพระโอวาทในป็อปอัป เพิ่มเป็นอย่างน้อย 4 ประโยค
→ ฝั่ง backend: `api/_lib/oracle.js` (ตอนนี้เลือกพารากราฟ 40–300 ตัวอักษร — ต้องขยาย/รวมหลายประโยค)

### 4. ผลลัพธ์ค้นหาไม่ตรงคำ (ความสำคัญ 2 — กระทบการทำงาน)
ไม่ต้องการ synonym expansion: ค้น "เมตตา" ห้ามไฮไลต์/แมตช์ "รัก", ค้น "ปัญญา" ห้ามแมตช์ "บำเพ็ญ/ศีล/สมาธิ/ปฏิบัติ", ค้น "เข้าใจไหม" ห้ามแมตช์ "ใจ/จิตใจ"
→ `api/_lib/passages.js` (`expandQueryTerms` + `keywordMap` ที่พอร์ตมาจากเว็บเดิม) + การไฮไลต์ใน `src/pages/Search.tsx`

### 5. ข้อมูล "สถานที่" ในหน้าผลค้นหาแสดงผิด (ความสำคัญ 3)
ภาพประกอบ: การ์ดผลค้นหาโชว์ "· วันนี้ · จ.แผ่นดินเกิดภัยพิบัติ..." — ประโยคเนื้อหาโผล่ในตำแหน่งสถานที่
→ สาเหตุน่าจะเป็นข้อมูล temple_th/province_th ในคลังสกปรก (มีทั้ง "จ.โคราช" vs "จ.นครราชสีมา", ค่าเป็นประโยคยาว) — ต้องตรวจ data + `placeLabel()` ใน `src/lib/format.ts`

### 6. [เสนอ] สถานที่/วันที่ว่าง ให้แสดง "ไม่ระบุ" แทนการซ่อน
→ `src/pages/Search.tsx`, `src/pages/FullText.tsx`, modal ใน `Home.tsx`

### 7. Header ทุกหน้าควรเหมือนกัน (ความสำคัญ 3)
- เพิ่มไอคอน user / ปุ่ม login ให้ครบทุกหน้า (ตอนนี้มีแค่หน้าหลัก — ภาพชี้มุมขวาบนว่างในหน้าค้นหา/ฉบับเต็ม)
- โครงสร้าง header จัดวางแบบเดียวกันทุกหน้า
→ ควรแยก component Header กลาง ใช้ร่วม 4 หน้า

### 8. เอาข้อความลิมิตใต้ปุ่มออก + ไม่ต้องจำกัดจำนวนครั้ง (ความสำคัญ 3)
ไม่ต้องแสดง "เปิดได้วันละ 2 ครั้ง · วันนี้เปิดแล้ว 1/2" และ "ผู้ดูแลระบบ · เปิดได้ไม่จำกัด"
**หมายเหตุ: ขัดกับสเปกเดิมที่สั่งให้จำกัด 2 ครั้ง/วัน — ต้องคอนเฟิร์มกับทีมก่อนแก้**
→ `src/pages/Home.tsx`

### 9. เปิดอ่านจากผลค้นหา → เปิดแท็บใหม่ + เอาเมนู "< กลับคลังพระโอวาท" ออก (ความสำคัญ 3)
→ `src/pages/Search.tsx` (target="_blank"), `src/pages/FullText.tsx` (ลบลิงก์กลับคลัง)

### 10. ลายน้ำอ่านไม่ออก — ปรับสีให้เห็นง่ายขึ้น (ความสำคัญ 3)
→ `src/pages/FullText.tsx` (`wmColor` ตอนนี้ opacity ~0.05)

## สิ่งที่เห็นเพิ่มจากภาพประกอบ (ไม่อยู่ในตาราง — ควรแจ้งทีม)
- **เนื้อหามีขยะ pandoc**: บางฉบับโชว์ `{width="4.83..in" height="4.83..in"}` ในเนื้อความ (ภาพหน้า จี้กงหัวฝอ ไท่เชิ่ง จ.อุดรธานี 15 ก.ย. 2562) → ควรกรองใน `splitIntoParagraphs`/ตอน export ข้อมูล
- **ผลค้นหาซ้ำ**: คำค้นเดียวเจอฉบับเนื้อหาเดียวกัน 2–3 การ์ด (คนละ id/ชื่อองค์สะกดต่างกัน เช่น "สงฆ์จี้กงวิปลาสแห่งซีหู" ×2 + "พระสงฆ์จี้กงวิปลาส") → ข้อมูลในคลังซ้ำซ้อน
- ชื่อจังหวัดไม่มาตรฐาน: "จ.โคราช" / "จ.นครราชสีมา" / "จ.กทม" ปนกัน → กระทบฟิลเตอร์สถานธรรม
